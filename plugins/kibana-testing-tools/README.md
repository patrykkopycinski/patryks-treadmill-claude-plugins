# Kibana Testing Tools

Comprehensive testing and QA automation for Kibana development.

## Skills (6)

### 🔍 @kbn-evals-debugger
Debug @kbn/evals test failures with OTEL trace analysis, adaptive convergence, and automatic fixes.

### 🔄 @cypress-to-scout-migrator  
Strategic test suite optimizer - migrates Cypress to Scout with quality improvements, not 1:1 conversion.

### 🐛 @flake-hunter
Systematic flaky test debugger with 5 root cause categories and 50-100 run verification protocol.

### 📊 @test-coverage-analyzer
Comprehensive test coverage analysis with gap detection and recommendations.

### 🧪 @api-test-generator
Generate Scout API tests from Kibana route definitions with RBAC, validation, and edge case coverage.

### 🏭 @test-data-builder
Mock data generation from TypeScript interfaces with ES archive creation and factory patterns.
