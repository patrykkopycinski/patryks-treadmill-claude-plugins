# Kibana Code Quality Suite

Automated code quality, security, and accessibility reviews with healing capabilities.

## Skills (5)

### 🩹 @type-healer
Automatic TypeScript error fixer with 10 error categories and zero-tolerance for suppressions.

### ♻️ @refactor-assistant
Safety-first refactoring with mandatory test baseline, metrics tracking, and rollback protocol.

### 🔐 @security-reviewer
Vulnerability scanner with validated test suite - detects XSS, SQL injection, auth bypass, and more.

### ♿ @accessibility-auditor
WCAG 2.1 compliance auditor using Scout/axe-core with keyboard navigation validation.

### 📚 @skill-curator
Skill ecosystem quality manager with MCP-powered similarity detection and health scoring.
